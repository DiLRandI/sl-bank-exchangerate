# sl-bank-exchange-rate-console

This is a console application which will display the exchange rate for **LKR** for most banks in Sri Lanka.

## supported banks

- [Bank Of Ceylon](https://boc.lk/index.php?route=rates/rates)
- [Hatton National Bank](https://www.hnb.net/exchange-rates)
- [Commercial Bank](https://www.combank.lk/rates-tariff#exchange-rates)
- [HSBC Sri Lanka (PDF)](https://www.hsbc.lk/content/dam/hsbc/lk/documents/tariffs/foreign-exchange-rates.pdf)
- [National Development Bank](https://www.ndbbank.com/rates-and-tariffs?active=exchange_rates)
